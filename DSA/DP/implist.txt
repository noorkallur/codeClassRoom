Here's a comprehensive list of 1D Dynamic Programming (DP) problems categorized by technique or pattern. These are essential for coding interviews and mastering them will give you a strong foundation in DP.

🧠 1. Basic Memoization / Tabulation
These problems help you understand the core idea of storing results to avoid recomputation.

Fibonacci Numbers
Climbing Stairs
Min Cost Climbing Stairs
House Robber
House Robber II (circular array)
Tribonacci Number
Decode Ways
Jump Game
Jump Game II
Maximum Subarray (Kadane’s Algorithm)



🔁 2. DP with State Transitions
These involve transitioning between states based on decisions.
Longest Increasing Subsequence
Number of Longest Increasing Subsequences
Partition Equal Subset Sum
Target Sum
Combination Sum IV
Coin Change
Coin Change II
Perfect Squares
Integer Break


🧩 3. Subset / Knapsack Variants
Classic 0/1 and unbounded knapsack problems.
0/1 Knapsack
Subset Sum
Count of Subset Sum
Minimum Subset Sum Difference
Target Sum (can be reduced to subset sum)
Partition Equal Subset Sum
Rod Cutting Problem
Unbounded Knapsack
🧵 4. String DP (1D optimized)
Some string problems can be optimized to 1D DP.

Longest Palindromic Substring (space optimized)
Longest Common Subsequence (space optimized)
Edit Distance (space optimized)
Distinct Subsequences
Wildcard Matching (space optimized)
🔄 5. Greedy + DP Hybrid
These problems combine greedy decisions with DP.

Jump Game II
Maximum Product Subarray
Best Time to Buy and Sell Stock (with cooldown, transaction fee, etc.)
📈 6. Monotonic DP / Convex Hull Trick (Advanced)
These are more advanced and often used in competitive programming.

Maximum Sum of 3 Non-Overlapping Subarrays
Minimum Cost to Merge Stones
Line of Sight / Convex Hull Trick problems
🧮 7. Digit DP (Advanced)
Used for counting numbers with certain properties.

Count numbers with unique digits
Count numbers with digit sum divisible by K
Count numbers with no adjacent same digits
🧠 8. Bitmask + 1D DP
Used when the state can be represented by a bitmask.

Count All Valid Pickup and Delivery Options
TSP (Traveling Salesman Problem) with bitmask
Minimum Cost to Connect Two Groups of Points
