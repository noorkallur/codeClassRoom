# Traversals Methods:
### 1. Pre-Order
Pre-order traversal is a tree traversal method where you print the root node first, then recursively print the left subtree, and finally the right subtree.
N -> L -> R

#### usages
1. Used for evaluating math expressions
2. making a copy of data
3. serializing

### 2. In-Order:
In-order traversal is a tree traversal method where you recursively visit the left most node print the left node first, then print the root node second, and finally print the right subtree.
L -> N -> R

#### usages
1. To traverse in BST in sorted order

### 3. Post-Order:
Post-order traversal is a tree traversal method where you recursively visit the left most node and print left first, then the right next, and finally the root node.
L -> R -> N 

#### usages
1. delete node in binary tree
2. Bottom up calculation
