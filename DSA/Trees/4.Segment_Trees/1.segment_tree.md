# Segment Tree
Segement tree is a binary tree which has interval and info for each node of an array

# When to use?
Perform query(sum, max, avg, min, prod) on a range

# Algorithm
## Build:
Start from the root and recursively build the left and right subtrees.
Combine the results of the left and right subtrees to form the parent node.

## Update at index:
If the current node's range is completely within the update range, add the value to the node.
If the current node's range is partially within the update range, propagate the update to the child nodes.
Recalculate the value of the current node based on its updated children.

## Query for range:
If the current node's range is completely within the query range, return the node's value.
If the current node's range is partially within the query range, query the child nodes.
Combine the results of the child nodes to get the final result.
